## Practical UE5 Execution Notes

- Court terme: privilégier `Integration + E2E smoke` (retour concret) plutôt qu'un investissement massif en tests unitaires UE.
- Build local standard: `-NoXGE` tant que XGE n'est pas stabilisé.
- Chaque migration BP->C++: backup, migration-check, compile, smoke PIE.
