# Politique de sécurité — Space League

## Signalement d'une vulnérabilité

Si vous identifiez une faille, une fuite potentielle d'information ou un problème de sécurité :

1. **Ne publiez pas** d'exploit ni de détail sensible en public
2. **Ouvrez une issue** uniquement si le sujet peut être discuté sans risque
3. Sinon, **contactez l'équipe** via GitHub Issues pour un traitement confidentiel

## Périmètre

Ce dépôt contient du code source, de la documentation et des configurations.
Il ne contient pas de données sensibles, de credentials ou d'informations personnelles.

Les assets complets du projet (Content UE5) sont hébergés sur Perforce, hors de ce dépôt.
